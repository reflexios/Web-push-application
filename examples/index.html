Web Push Demo

A minimal client for subscribing to browser push notifications and sending a test push through your platform.

Set your credentials in server.py: PLATFORM_URL, CLIENT_ID, API_KEY, VAPID_PUBLIC_KEY. The API key never reaches the browser — it stays on the server.
